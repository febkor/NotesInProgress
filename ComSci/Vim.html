Vim

Modes

  • normal/command mode
  • insert mode
  • visual mode

Insert mode

To enter from command mode:

  • i : insert before current position
  • a : insert after current position (append)
  • I : insert at first non-blank char in line
  • A : insert at last char of the line

Normal mode

Hit <Esc> to enter this mode.

Grammar

{operator}{count}{object}
object = {motion, text object}

Operators

  • . : repeat the last command
  • y : yank/copy
  • c : change; cut and enter insert mode
  • C : change the rest of the current line
  • d : delete; cut but remain in normal mode
  • D : delete the rest of the current line
  • p : paste after the cursor
  • P : paste before the cursor
  • o : insert new line after
  • O : insert new line before

Single char operators (can’t modify with count)

  • x : delete character under the cursor
  • X : delete characters before the cursor
  • r : replace char under cursor with another char
  • s : substitute char under cursor and move to insert mode

Misc

  • U : undo
  • Ctrl-R : redo
  • g~ : toggle case
  • gU : uppercase
  • gu : lowercase

Motions

      ↑
← h j k l →
    ↓

Word motions

  • w / b : jump word to word
  • e / ge: jump to end of word
  • W / B : jump WORD to WORD
  • E / gE: jump to end of WORD

A word is alphanumeric. A WORD is a word and punctuation .({}),.

Horizontal motions

  • 0 : the first char of a line
  • ^ : the first non-blank char of a line
  • $ : the last char of a line
  • g_ : the last non-blank char of a line

Vertical motions

  • gg : go to the top of the file
  • {line}gg : go to a specific line
  • G : go to the end of the file
  • } : jumps entire paragraphs downwards
  • { : similarly but upwards
  • CTRL-D : move down half a page
  • CTRL-U : move up half a page
  • H : top line on screen
  • M : middle line on screen
  • L : bottom line on screen

Find character motions

  • f{char} : find next occurrence of char in a line
  • F{char} : backwards
  • t{char} : move cursor to before the next occurrence of char
  • ; (next) and , (prev): repeat the last char search

Find pattern motions

  • /{pattern} : to search forward inside a file
  • ?{pattern} : to search backwards
  • n : jump to next match
  • N : jump to prev match
  • /<Enter> or ?<Enter>: run the last search
  • * : to search for the next word under the cursor
  • # : to search for the prev word under the cursor
  • % : jump to matching ({[]})
  • gd : jump to definition of whatever is under your cursor
  • gf : jump to a file in an import

Text object

Adjectives

  • i : inside
  • a : around, inside and surrounding ({[ ]})

Nouns

  • l : letter
  • w : word
  • W : WORD
  • s : sentence
  • p : paragraph
  • t : tag - xml, html
  • ( : ()
  • { : {}
  • < : <>
  • " : ""
  • ' : ’’

Some operators (ydc) can be doubled (e.g. yy) to make them operate on a whole line.

Visual mode

  • v : switch to visual mode (character oriented)
  • V : switch to visual mode (line oriented)
  • Ctrl-V : switch to block-visual mode (select rectangles of text)

Once the selection has been made, you can:

  • J : join all the lines together.
  • < : indent to the left
  • > : indent to the right
  • = : auto indent

Advanced

Registers

"{register name}{operator}{count}{object}

write to named registers: a-z append to named registers: A-Z _: black hole register (not cut)

:registers (alias :reg) prints contents of all registers :reg a prints contents of register a

Program commands

  • :e <path/to/file> : open
  • :w : save
  • :saveas <path/to/file> : save to <path/to/file>
  • :x, ZZ or :wq : save and quit (:x only save if necessary)`
